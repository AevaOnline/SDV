sample_all(num_rows=5)[source]¶

Samples the entire database.

Parameters:num_rows (int) – Number of rows to be sampled on the parent tables.
Returns:Tables sampled.
Return type:dict

sample_all returns a dictionary with all the tables of the dataset sampled. The amount of rows sampled will depend from table to table, and is only guaranteed to match num_rows on tables without parents.

This is this way because the children tables are created modelling the relation thet have with their parent tables, so it’s behavior may change from one table to another.
